Step 3: Linear Regression from Scratch:

The Cost Function Linear regression works by finding the "best fit" line for our data. But what exactly does "best fit" mean? It's the line that minimizes the difference between the actual data points and the points on our predicted line.

To measure this difference, we use a cost function (also known as a loss function). A common one is the Mean Squared Error (MSE). The MSE is the average of the squared differences between the predicted values and the actual values.

$$ MSE = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2 $$

Where:

$$ y_i : \text{ actual value} $$

$$ \hat{y}_i : \text{ predicted value from the model} $$

$$ n : \text{ number of data points} $$

Let's write a simple function to calculate the MSE for any given slope (m) and y-intercept (b).

In[5]:
def mean_squared_error(m, b, X, y):
    """Calculates the mean squared error for a given slope and intercept."""
    y_predicted = m * X + b
    mse = np.mean((y - y_predicted)**2)
    return mse

X = df['Attendance_Hours'].values
y = df['Final_Marks'].values

To see this in action, let's experiment with a few different lines and see how their MSE changes. Our goal is to find the line that gives us the lowest MSE.

In[6]:
# Experiment 1: A good guess for a line.
m1, b1 = 0.5, 40

Our goal is to find the m and b that give us the absolute minimum MSE. While we can do this manually with more complex algorithms like Gradient Descent, libraries like scikit-learn automate this process for us.

Step 4: Building the Model with Scikit-learn

Now that we understand the underlying principle, let's use a professional library to find the optimal m and b for our data. scikit-learn is a powerful machine learning library that makes this process very efficient.

In[7]:
from sklearn.linear_model import LinearRegression

# Prepare the data. The input variable (X) must be a 2D array.
X_2d = df[['Attendance_Hours']].values
y = df['Final_Marks'].values

# Create and fit the linear regression model.
model = LinearRegression()
model.fit(X_2d, y)

# Print the model's coefficients.
print(f"The optimal slope (m) is: {model.coef_[0]:.4f}")
print(f"The optimal y-intercept (b) is: {model.intercept_:.4f}")

The optimal slope (m) is: 0.4889
The optimal y-intercept (b) is: 40.9845

Scikit-learn finds the values of m and b that give the lowest MSE. The slope m shows how much the final mark increases for each extra hour of attendance, while the intercept b is the expected mark when attendance is zero.

Step 5: Making a Prediction

Now that our model is trained, we can use it to make predictions. Let's see what marks a student might get if they attend a certain number of hours.

In[8]:
# Let's predict the final marks for a student with 45 hours of attendance.
hours_to_predict = np.array([[45]])

Step 6: Visualizing the Regression Line

Finally, let's put it all together and visualize our final model. We will plot the original data points and then add the regression line scikit-learn found for us. This plot will clearly show how well our linear model fits the data.

In[9]:
